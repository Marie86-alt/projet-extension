# Extension de Traduction pour Chrome

## Description

L'Extension de Traduction pour Chrome est une extension légère et pratique qui vous permet de traduire instantanément du texte dans votre navigateur Chrome. 
Que vous naviguiez sur des pages Web étrangères ou que vous ayez besoin de traduire du contenu pendant votre lecture, cette extension vous offre une solution simple et rapide pour comprendre et communiquer dans différentes langues.

## Fonctionnalités

- **Traduction automatique** : La langue à traduire est automatiquement détectée, vous évitant ainsi de devoir spécifier la langue source.
- **Choix de la langue cible** : Vous pouvez choisir la langue vers laquelle vous souhaitez traduire le texte.
- **Accès rapide à la page de traduction Google** : Vous pouvez également accéder à la page de traduction Google pour une traduction plus détaillée.
- **Utilisation de l'API libre Translate** : L'extension utilise l'API libre Translate pour effectuer les traductions de manière efficace et précise.
- **Conteneur Docker** : Pour faciliter le déploiement et la gestion de l'API, nous utilisons Docker pour encapsuler l'environnement de développement.

## Langues prises en charge

L'extension prend en charge la traduction vers 44 langues, couvrant une grande variété de langues parlées dans le monde entier. Parmi les langues prises en charge, on trouve :

- Anglais
- Français
- Espagnol
- Allemand
- Chinois (simplifié et traditionnel)
- Japonais
- Et bien d'autres !

## Installation

Pour installer l'Extension de Traduction pour Chrome, suivez ces étapes simples :

1. Téléchargez l'extension à partir du Chrome Web Store.
2. Cliquez sur "Ajouter à Chrome" pour installer l'extension.
3. Une fois installée, l'extension sera prête à être utilisée. Vous verrez une icône de traduction dans votre barre d'outils Chrome.
